# COA Labs (English Web Edition)

This repository contains the English web edition of the lab manual for the book 《计算机组成原理实验指导》.

The site is built with Astro and Starlight.

## Development

This project uses pnpm.

- Install dependencies: `pnpm install`
- Start dev server: `pnpm dev`
- Build: `pnpm build`
- Preview the build: `pnpm preview`

## Content

Documentation pages live under `src/content/docs/`. The generated site is written to `dist/`.
