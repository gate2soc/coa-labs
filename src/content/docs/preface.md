This book uses formatting and notation with specific meanings:

- **Bold text** highlights parts that require special attention. These are critical for understanding key concepts or completing the current lab steps.
- <span style="color: var(--sl-color-green-high); font-weight: 600;">Green terms</span> indicate **specialized terminology**, often originating from lab software, official documentation, or specifications. If the original term is English, this manual uses the commonly accepted translation and shows the original term at its first appearance.
- `Monospace text` is used for code blocks, file paths, commands, and links that should be copied verbatim. When possible, syntax highlighting is provided.
- `A[N-1:0]` denotes an **N-bit** binary signal.
